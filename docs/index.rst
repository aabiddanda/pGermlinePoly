Welcome to pGermlinePoly's documentation!
==========================================

``pGermlinePoly`` is a Bayesian model to estimate the posterior probability of
germline polymorphism in somatic sequencing data. It implements an EM algorithm
that jointly estimates logistic annotation weights and a Beta-Binomial error
concentration to discriminate germline heterozygotes from somatic variants.
